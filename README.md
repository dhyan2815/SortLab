## How It Works

1. User selects a **sorting algorithm** from the dropdown.  
2. Inputs a list of **numbers** (comma-separated).  
3. On clicking **Sort Data**, the visualization animates each step of the chosen algorithm.  
4. The site measures **execution time** and **memory usage** in real time.  
5. A comparison table and flowchart are **dynamically** generated.  
6. Results can be downloaded as a **CSV file** for analysis.
